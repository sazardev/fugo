# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| latest  | ✅ Yes    |

## Reporting a Vulnerability

To report a vulnerability, please open a [private security advisory](https://github.com/sazardev/fugo/security/advisories/new) on GitHub.

You can expect:
- **Confirmation** within 48 hours of submission
- **Updates** every 7 days until resolution
- **Credit** in the advisory and CHANGELOG once the issue is resolved

If you do not receive a response within 72 hours, please reach out via the repository's discussions page as a fallback.

Please do **not** report security vulnerabilities via public GitHub issues or pull requests.
